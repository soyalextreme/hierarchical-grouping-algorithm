Algoritmo de agrupamiento por distancias minimas.
Genera un numero de puntos aleatorios en un plano y los ordena.
El conjunto de los puntos los va guardando en clusters.
Lo muestra de forma grafica con paginas de HTML gracias a la libreria Bokeh
CUIDAD el echo de que grafica cada una de las iteraciones de el agrupemiento 
puede hacer que tu computador se trabe.


PASOS PARA CORRER EL ALGORITMO:
1-. Clona el repositorio con git clone <url del proyecto>
2-. Crea y ejecuta tu entorno virtual de python 
        *python3 -m venv enc
        *source env/bin/activate
3-. installa bokeh en el entorno virtual    
        *pip install bokeh
4-. corre el archivo main.py 
        python3 main.py

Cuidado con el numero de puntos aleatorios que generas.
